Supported OS: Windows, Linux

Description

By default Xeneth is configured to scan COM ports 1 to 4 of the computer. In some cases the user has more ports or is unable to assign the port in question to a lower range. It is possible to alter these default scan ranges by changing the device discovery properties prior to calling the enumeration function.

To enumerate these properties some utility functions have become available. These property system functions are also prepended with "XCD_" but operate the same as the camera object property system utility functions. For more information about enumerating properties refer to the SDK help file under "Samples -> C/C++ Samples -> Property system -> Enumerate properties".

Used functions

The following table lists the functions used in this sample:

Function Description
XCD_GetPropertyValueL Gets an integer typed value from the discovery properties system
XCD_SetPropertyValueL Sets an integer typed value in the discovery properties system
