Supported OS: Windows, Linux

Description

The window of interest (or WOI in short) is configured by making use of the property system. It allows the user to get higher framerates by decreasing the time needed to process one frame, and thus also reducing the frame size.

When the new window is set make sure that the buffer which holds the last retrieved frame is adjusted to accomodate the new framesize.
It is good practice to check the values of the properties after the WOI has changed because the given values can be altered internally in order to fit the window's constraints.
Note: Each camera has their own constrains for specifying a window. To make sure which constraints the camera has, check the camera specific WOIxx properties. These ranges can also be retrieved using the XC_GetPropertyRange function, which is demonstrated in the "Enumerate Properties"-sample.

The following properties are generic for all cameras that are capable of setting a window of interest:

Property name Description
WoiSX(0) Defines the X coordinate of the start of the window of interest.
WoiEX(0) Defines the X coordinate of the end of the window of interest.
WoiSY(0) Defines the Y coordinate of the start of the window of interest.
WoiEY(0) Defines the Y coordinate of the end of the window of interest.

To check if the camera supports window of interest retrieve the property "CapWoiCount", if this property exists the XC_GetProperty function returns I_OK.

Used functions

The following table lists the functions used in this sample:

Function Description
XC_OpenCamera Opens a connection to the camera specified and returns a handle pointing to the camera. The handle is used in all other functions so keep a reference.
XC_CloseCamera Closes and cleans up the connection to the camera specified by the handle which was obtained by calling the XC_OpenCamera function.
XC_IsInitialised Check if the connection was successfully initialized.
XC_StartCapture Start capturing frames.
XC_StopCapture Stop capturing frames.
XC_IsCapturing Check if the device is capturing frames.
XC_GetFrameSize Used to determine the native frame size.
XC_GetPropertyValueL Retrieve a long value from the camera specified by the propertyname.
XC_SetPropertyValueL Sends a long value to the camera specified by the propertyname.
