Supported OS: Windows, Linux

Description

In this samples all functions for retrieving en writing properties are demonstrated. The count of all properties is requested, base on this count every property is enumerated with specific details about this property.

Used functions

The following table lists the functions used in this sample:

Function Description
XC_OpenCamera Opens a connection to the camera specified and returns a handle pointing to the camera. The handle is used in all other functions so keep a reference.
XC_IsInitialised Check if the connection was successfully initialized.
XC_CloseCamera Closes and cleans up the connection to the camera specified by the handle which was obtained by calling the XC_OpenCamera function.
XC_GetPropertyCount Gets the number of properties.
XC_GetPropertyName Gets the property name.
XC_GetPropertyCategory Gets the property category.
XC_GetPropertyType Retrieves the type of a property
XC_GetPropertyRange Retrieves the boundaries of a property For an enumerated type this is a list of possible values, for numeric types it returns "lowvalue>highvalue".
XC_GetPropertyUnit Retrieve units supported by this property, default unit comes first.
XC_GetPropertyValue Retrieve a value from the camera specified by the propertyname.
XC_GetPropertyValueL Retrieve a long value from the camera specified by the propertyname.
XC_GetPropertyValueF Retrieve a double from the camera specified by the propertyname.
XC_GetPropertyBlob Retrieve a binary large object (blob) from the camera, this is usually Firmwire, calibratoin data or overlaydata
