Supported OS: Windows, Linux

Description

In this sample we demonstrate how to configure and use a Lynx GigE Vision (F085) camera for software triggering.

The first part of the sample establishes a connection and makes sure we are connected to the correct camera by checking the camera's product id (PID). If the correct camera was connected we continue by configuring the trigger control in software triggered mode. Since the software trigger does not rely on trigger sensitivity and/or trigger polarization it is only necessary to configure to camera in a triggered mode, frame or line.

This sample will configure the camera in line triggered mode and generate the amount of software triggers needed to capture one frame. For every time a software line trigger is executed the camera will integrate a new line to add to the final image. This continues until all lines currently configured for one frame have been triggered after which the frame becomes available in the SDK.

Setup software triggered mode

Property Value Description
TriggerMode Line: SMA - Frame: Free running.
Value is 5 for F083, 2 for F085. 		The trigger mode configures which trigger signal is triggered on activation, the line or the frame trigger.
LineTriggerEnable Off (0) Enable / Disable the line trigger block.
FrameTriggerEnable Off (0) Enable / Disable the frame trigger block.

Execute software trigger

For each line in a frame a software trigger is executed.

Property Value Description
SoftwareLineTrigger Execute (1) Execute a software trigger.

Used functions

The following table lists the functions used in this sample:

Function Description
XC_OpenCamera Opens a connection to the camera specified and returns a handle pointing to the camera. The handle is used in all other functions so keep a reference.
XC_CloseCamera Closes and cleans up the connection to the camera specified by the handle which was obtained by calling the XC_OpenCamera function.
XC_IsInitialised Check if the connection was successfully initialized.
XC_IsCapturing Check if the SDK is capturing frames.
XC_StartCapture Start capturing frames.
XC_StopCapture Stop capturing frames.
XC_GetFrame Used to grab a raw frame from the device.
XC_GetFrameSize Used to determine the native frame size.
XC_GetMaxWidth Used to grab a raw frame from the device.
XC_GetMaxHeight Used to determine the native frame size.
XC_SetPropertyValueL Sets an integer typed value in the discovery properties system.
XC_GetPropertyValueL Gets an integer typed value from the discovery properties system.
XC_ErrorToString Converts an error code to a string.
