Supported OS: Windows, Linux

Description

This sample applies to the Tigris camera.

It shows how to exploit the on-board radiometric data to compute the temperature of observed objects.
The two available thermography modes are illustrated.
In the temperature mode, temperatures are calculated via linear interpolation.
In the radiometric mode, temperatures can be obtained as a result of applying a dedicated analytic function.

The sample assumes that a radiometric calibration is loaded in the selected image correction source on board of the camera. Note that this sample does not apply to the Serval camera.

Used functions

The following table lists the functions used in this sample:

Function Description
XC_OpenCamera Opens a connection to the camera specified and returns a handle pointing to the camera. The handle is used in all other functions so keep a reference.
XC_CloseCamera Closes and cleans up the connection to the camera specified by the handle which was obtained by calling the XC_OpenCamera function.
XC_IsInitialised Check if the connection was successfully initialized.
XC_StartCapture Start capturing frames.
XC_StopCapture Stop capturing frames.
XC_IsCapturing Check if the device is capturing frames.
XC_GetFrame Used to grab a raw frame from the device.
XC_GetFrameSize Used to determine the native frame size.
XC_GetMaxValue Retrieve the maximum allowed ADU value.
XC_GetWidth Used to determine the width of the camera.
XC_GetHeight Used to determine the height of the camera.
XC_GetPropertyValueF Retrieve a floating point value from a camera feature specified by a property name.
XC_SetPropertyValueF Modify a camera feature with a specified property name by writing a floating point value.
XC_SetPropertyValueE Modify a camera feature with a specified property name by writing an enumeration string value.
